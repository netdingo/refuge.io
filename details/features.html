Blob Server

A blob server allows you to store and retrieve blobs, a blob being a set of bytes.

  • Multi-backend (FS, Distributed FS, S3...)
  • Simple REST API: GET,PUT, POST, DELETE
  • Synchronisation between containers
  • All blobs are uniquely identitified (content-hash)
  • Works with mobile & embedded devices
  • Partials upload & downloads supported
  • RESTFUL, resources visible on HTTP

Refuge contains a blob server named "Coffer" (see here for more details).
